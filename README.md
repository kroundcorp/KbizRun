# K-Biz Run (kbizrun.com)

공공조달관리사 국가자격증 LMS 플랫폼 — (주)케이비즈글로벌교육원

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

개발 서버: http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## 주요 스크립트

- `npm run dev` — 개발 서버 실행
- `npm run build` — 프로덕션 빌드
- `npm run preview` — 빌드 결과 미리보기
- `npm run lint` — TypeScript 타입체크
