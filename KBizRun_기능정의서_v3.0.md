### 3.1.1 영상 중간 퀴즈 (In-Video Quiz) [➕ 신규 A12]

영상 강의 시청 중 핵심 개념 확인을 위해 특정 타임스탬프에서 퀴즈가 자동 출제되는 인터랙티브 학습 기능.
상세 설계 문서: `video-quiz.md`

**핵심 개념**

| 용어 | 설명 |
|---|---|
| 큐포인트(Cue Point) | 영상 내 타임스탬프와 문제를 연결하는 트리거 지점. 형식 `timestampSec` (초 단위) |
| 인터럽트 방식 | `hard` (필수 풀이) / `soft` (건너뛰기 가능) / `overlay` (미니 퀴즈) 3종 정의. **Phase 1은 `hard` 만 구현** |
| 필수 통과(is_required) | true 이면 반드시 응답해야 재생 재개. false 이면 "건너뛰기" 노출 |

**데이터 모델** (`src/data/videos.ts`)

```ts
interface VideoCuePoint {
  id: string;
  timestampSec: number;
  interruptType: 'hard' | 'soft' | 'overlay';
  isRequired: boolean;
  quiz: VideoQuiz;  // 4지선다 + 정답 + 해설 + 배점
}

interface VideoLecture {
  id; subjectId; index; title; instructor;
  durationSec; videoUrl; description;
  cuePoints: VideoCuePoint[];
}
```

**학습자 UX 플로우**

1. 영상 재생 중 `timeupdate` 이벤트로 현재 시간 모니터링
2. 완료되지 않은 큐포인트의 `timestampSec`에 도달하면 영상 자동 일시정지
3. `VideoQuizModal` 노출 (질문 + 4지선다 + 필수/권장 뱃지)
4. 정답 제출 → 정답/오답 및 해설 표시 (정답 시 배점 +10pt 표시)
5. "계속 시청하기" 클릭 시 영상 자동 재개
6. 완료 큐포인트는 sessionStorage에 기록되어 되감기 시 재출제 안 됨

**관리자/강사 기능** [⏸ Phase 2]

- 큐포인트 설정기 (타임라인 드래그앤드롭), 기존 문제 은행 연결, AI 자동 생성
- 큐포인트별 정답률·응답 시간·건너뛰기 비율 대시보드
